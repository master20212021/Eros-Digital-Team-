# D'FRANCO HAIR SALON — Website

Professional website for **D'Franco Hair Salon NY** — Full Service Salon & Master Colorist located in Woodside, Queens, NY.

## Live Site

🌐 **[View Website](https://master20212021.github.io/D-FRANCO-HAIR-SALON-/)**

## Features

- Elegant, professional single-page design
- Mobile-first responsive layout
- Smooth scroll animations (AOS library)
- Service showcase with 6 categories
- Client testimonials slider
- Team showcase section
- Contact form with floating labels
- Google Maps integration
- Instagram gallery section
- Newsletter subscription
- Floating call-to-action button
- Custom brand colors from storefront

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- [AOS](https://michalsnik.github.io/aos/) — Animate on Scroll
- [Font Awesome 6](https://fontawesome.com/) — Icons
- Google Fonts (Cormorant Garamond + Montserrat)

## Deployment

This site is deployed via **GitHub Pages** from the `main` branch.

### To deploy:
1. Push all files to the repository
2. Go to Settings → Pages
3. Set source to `main` branch, root folder
4. Site will be live at the URL above
