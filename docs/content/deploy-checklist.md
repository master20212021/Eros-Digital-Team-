# Checklist de despliegue

## GitHub
- Crear el repositorio final o confirmar la URL del repo ya creado.
- Conectar el repo local con `origin`.
- Hacer primer commit y push.

## Hosting
- GitHub Pages ya esta configurado en el repo.
- Verificar en `Settings > Pages` que el dominio custom quede asociado.
- Esperar a que GitHub emita SSL para el dominio una vez propaguen los DNS.

## Dominio en GoDaddy
- Entrar al panel DNS de `erosdigitalteam.com`.
- Añadir los registros que indique el hosting.
- Apuntar el dominio raiz y `www`.
- Esperar propagacion DNS.

## Validaciones finales
- Revisar version mobile y desktop.
- Probar CTA de contacto.
- Confirmar metadatos, favicon y enlaces.
- Comprobar SSL activo.
